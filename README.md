# myplan-app
A student academic planning app written in React. This app was created using Create React App.

### Prerequisites
* Node
* ReactJS

### Installation

Clone this repository:
```
  $ git clone https://github.com/charlon/myplan-app.git
  $ cd myplan-app
```

Install Node dependencies:
```
  $ npm install
```
Start the server:
```
  $ npm start
```
